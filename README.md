# TaskMe App

A productivity and task management application built with Flutter.

## Overview
TaskMe is a comprehensive task management tool designed to help users track and organize their activities efficiently. The app features multiple screens for different functionalities, including home, settings, statistics, and a timer.

## Features

- **Task Tracking**: Create and manage tasks with ease
- **Statistics**: View detailed statistics of your progress
- **Customization**: Adjust settings to tailor the app to your preferences
- **Timer Functionality**: Track time spent on tasks
- **Notifications**: Receive reminders and updates

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the app:
   ```bash
   flutter run
   ```

## Dependencies

- Flutter
- Firebase (for authentication and data storage)
- Provider (for state management)

## Configuration

Update the configuration files as needed:
- `firebase_options.dart`
- `pubspec.yaml`
